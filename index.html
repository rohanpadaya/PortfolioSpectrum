Projects


Tennis Match Prediction Using Machine Learning Algorithms

The project aimed to predict the outcomes of men’s tennis matches using machine learning algorithms. Using Python, data was cleaned and features were engineered based on ATP rankings, significantly improving model performance by 20%. Multiple models, including the ELO model, were constructed, achieving up to 63.78% accuracy, which was a 15% improvement over baseline models. The results were compiled into a LaTeX report, providing insights for potential future improvements.

Image Classification using Convolutional Neural Networks (CNN)

This project focused on classifying images from the CIFAR-10 dataset using deep learning techniques. Convolutional Neural Network (CNN) models were developed in TensorFlow and Keras, and data augmentation was applied to enhance model accuracy from 70.04% to 73.45%. Advanced techniques, such as hyper-parameter tuning and a ResNet50 architecture, further improved validation accuracy to 80.66%, resulting in a 15% improvement over the baseline.

Stock Price Prediction using Recurrent Neural Networks (RNN)

This project aimed to predict stock prices using Recurrent Neural Network architectures like Vanilla RNN, LSTM, and GRU. The models were successful in achieving low Mean Squared Errors (MSE) for Apple (0.0003329) and Samsung (0.0003621), reducing prediction error by 25% compared to previous models. Techniques such as dropout and batch normalization were used to enhance generalization, resulting in a 30% increase in model stability.


Breast Cancer Detection and Classification Using Machine Learning:

The project addressed the classification of malignant and benign breast cancer cases using histological image data. Various machine learning models, including Support Vector Machines and Decision Trees, were implemented and optimized to achieve a recall of over 95%. Feature engineering and data visualization led to a 20% improvement in accuracy, and the best model was selected based on precision and recall metrics. This project provided valuable insights into improving diagnostic accuracy for breast cancer detection.

Customer Segmentation for a Bank’s Credit Card Holders

This project utilized clustering algorithms, including K-Means and DBSCAN, to segment the bank’s credit card holders based on their spending behavior. Feature scaling, PCA, and cluster profiling were employed to identify distinct customer groups, supporting targeted marketing strategies for enhanced customer engagement.
